Luminous v0.60.0 contains exciting new features, design enhancements, and stability improvements since v0.50.0:

### 🌟 New Features
- **Home Discovery & Personalized Curation Hub**: A new landing dashboard featuring personalized curation queries, recently played tracks, and custom item sorting.
- **Artist Detail View**: A fully custom detail view for artists with a hero collage, discography, and playlist integrations, plus cover-flow style album art stacks on Artist cards.

### 🎨 Visual & Theme Customizations
- **Dynamic Color Palette Generation**: Analyzes cover artwork dynamically using a Median Cut algorithm and archetype scoring to extract dominant colors and apply them across the player interface.
- **Floating Glass PlayDock**: A modern, minimized floating music player deck with blur backdrops and theme-wide glassmorphism styling.
- **System Auto-Theme Integration**: Full support for light/dark mode syncing and matching brand accents.
- **Logo Visualizer Improvements**: Maps bass frequencies to the inner corona and treble to the outer ring for more responsive, dynamic SVG pulsing, with fixes to prevent glow clipping.

### 🐛 Stability & Bug Fixes
- **Rapid Skip Protection**: Prevents potential UI freeze when rapidly skipping tracks.
- **Mock Data Updates**: Upgraded screenshot utility mock data, cover arts, and sidebar states to ensure consistent presentation.
